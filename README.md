In this day and age AI has taken over much of the internet, making it hard to find a place to safley share somethign and not have AI steal it. This project is meant to fix that, by stopping bots in their tracks and making sure the humans do not use AI.

There are plently of things that block some bots, but this is meant to block ALL bots, and prevent all use of AI. It would require a simple application, easy for humans but hard for AI. It would take some work from the humans, but that is the price we pay.

The final vision looks like this: A site would redirect here with a (encripted) URL header saying where it came from. This site would have the user enter their pass phrase. Then, based on that URL header, the site woudl redirect to the protected link, and at the same time send a message to the protected site's server that the user is infact verified.
